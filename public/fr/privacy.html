Politique de confidentialité

Informations sur le traitement de vos données

1. Protection des données en bref

Informations générales

Les indications suivantes donnent un aperçu simple de ce qui arrive à vos données personnelles lorsque vous visitez ce site web. Les données personnelles sont toutes les données avec lesquelles vous pouvez être personnellement identifié.

Qui est responsable ?

Le traitement des données sur ce site web est effectué par l'exploitant du site (OYSI GmbH). Vous trouverez les coordonnées dans la section 3 de cette politique de confidentialité.

Comment collectons-nous vos données ?

Vos données sont collectées d'une part lorsque vous nous les communiquez (par exemple par e-mail ou formulaire de contact). D'autres données sont collectées automatiquement ou avec votre consentement lors de la visite du site web par nos systèmes informatiques (par exemple données techniques comme le navigateur, le système d'exploitation, l'adresse IP).

À quoi utilisons-nous vos données ?

Une partie des données est collectée pour garantir une mise à disposition sans erreur du site web. D'autres données peuvent être utilisées pour analyser votre comportement d'utilisateur – mais uniquement si vous y avez consenti.

2. Hébergement

Nous hébergeons le contenu de notre site web chez le prestataire suivant :

OVH

Le prestataire est OVH GmbH, Dudweiler Landstraße 5, 66123 Sarrebruck, Allemagne (ci-après « OVH »). OVH est un fournisseur cloud européen avec des centres de données dans l'UE.

Lorsque vous visitez notre site web, vos données personnelles sont traitées sur les serveurs d'OVH. Il peut s'agir notamment d'adresses IP, de demandes de contact, de métadonnées et de données de communication, d'accès au site web et d'autres données générées via un site web.

OVH dispose de certifications ISO 27001 et ISO 27701, garantissant ainsi un niveau élevé de protection des données. Les données sont traitées exclusivement dans des centres de données au sein de l'Union européenne.

Base juridique : L'utilisation d'OVH est basée sur l'art. 6 al. 1 lit. f RGPD. Nous avons un intérêt légitime à une présentation aussi fiable que possible de notre site web. Si un consentement correspondant a été demandé, le traitement est effectué exclusivement sur la base de l'art. 6 al. 1 lit. a RGPD.

Nous avons conclu un contrat de traitement des données (CTD) avec OVH. Il s'agit d'un contrat requis par la loi sur la protection des données, qui garantit qu'OVH traite les données personnelles des visiteurs de notre site web uniquement selon nos instructions et dans le respect du RGPD.

4. Collecte de données sur ce site web

Cookies et stockage local

Notre site web utilise des cookies et des techniques de stockage local. Ce sont de petits fichiers texte qui sont stockés sur votre appareil. Nous utilisons des technologies techniquement nécessaires pour garantir les fonctions du site web (Essentiels). Nous n'utilisons d'autres technologies (par exemple de Google) que si vous avez donné votre consentement dans la bannière de cookies.

Cookie Finalité Durée de stockage
oysi_consent Enregistre vos préférences de cookies 1 an
_ga, _ga_* Google Analytics (uniquement avec consentement) 2 ans

Base juridique : L'utilisation de technologies nécessaires est basée sur le § 25 al. 2 n° 2 TDDDG (loi allemande). Le traitement à des fins de marketing/analyse est basé sur votre consentement conformément à l'art. 6 al. 1 lit. a RGPD et § 25 al. 1 TDDDG.

Vous pouvez modifier vos paramètres des cookies à tout moment.

Fichiers journaux du serveur

Le fournisseur des pages collecte et stocke automatiquement des informations dans des fichiers journaux du serveur, que votre navigateur nous transmet automatiquement. Il s'agit de :

  • Type et version du navigateur
  • Système d'exploitation utilisé
  • URL de référence
  • Nom d'hôte de l'ordinateur accédant
  • Heure de la demande du serveur
  • Adresse IP (éventuellement anonymisée)

Ces données ne sont pas fusionnées avec d'autres sources de données. La base est notre intérêt légitime à la stabilité technique et à la sécurité du site web (art. 6 al. 1 lit. f RGPD).

Contact par e-mail ou téléphone

Si vous nous contactez par e-mail ou téléphone, votre demande, y compris toutes les données personnelles qui en résultent (nom, demande), sera stockée et traitée chez nous aux fins du traitement de votre demande. Nous ne transmettons pas ces données sans votre consentement. Le traitement de ces données est basé sur l'art. 6 al. 1 lit. b RGPD, si votre demande est liée à l'exécution d'un contrat ou est nécessaire à l'exécution de mesures précontractuelles.

5. Outils d'analyse (Google)

Google Consent Mode V2 (implémentation de base)

Nous utilisons le Google Consent Mode V2 dans l'implémentation de base (Basic Implementation). Ce service nous permet de transmettre à Google votre décision prise dans la bannière de cookies.

Consentement accordé : Google Analytics collecte et traite les données dans leur intégralité.

Consentement refusé : Aucune donnée n'est transmise à Google et aucun cookie n'est défini.

Le Consent Mode V2 contrôle les paramètres suivants, qui sont définis par défaut sur « denied » (refusé) :

  • ad_storage : Stockage de cookies publicitaires
  • ad_user_data : Transmission de données utilisateur à des fins publicitaires
  • ad_personalization : Publicité personnalisée
  • analytics_storage : Stockage de cookies d'analyse

Google Analytics 4

Si vous avez donné votre consentement, nous utilisons Google Analytics 4, un service d'analyse web de Google Ireland Limited, Gordon House, Barrow Street, Dublin 4, Irlande. Google Analytics utilise des technologies qui permettent la reconnaissance de l'utilisateur à des fins d'analyse du comportement de l'utilisateur (par exemple, cookies ou empreinte digitale de l'appareil).

Les informations collectées par Google sur votre utilisation de ce site web sont généralement transférées à un serveur de Google aux États-Unis et y sont stockées. Cependant, en activant l'anonymisation IP, votre adresse IP est préalablement tronquée par Google dans les États membres de l'Union européenne ou dans d'autres États parties à l'Accord sur l'Espace économique européen.

Transfert de données vers les États-Unis : Google base le transfert de données vers les États-Unis sur le cadre de protection des données UE-États-Unis (DPF). La Commission européenne a déterminé le 10 juillet 2023 que les États-Unis offrent un niveau adéquat de protection des données, à condition que l'entreprise soit certifiée. Google LLC est certifié selon le DPF.

Plus d'informations : Politique de confidentialité de Google

Base juridique : Le traitement est effectué exclusivement sur la base de votre consentement (art. 6 al. 1 lit. a RGPD). Vous pouvez révoquer votre consentement à tout moment via les paramètres des cookies.

6. Droits des personnes concernées

Conformément aux dispositions légales applicables, vous avez à tout moment le droit de :

  • Accès (art. 15 RGPD) : Vous avez le droit d'obtenir des informations sur vos données personnelles stockées chez nous.
  • Rectification (art. 16 RGPD) : Vous avez le droit de rectifier des données inexactes ou de compléter des données incomplètes.
  • Effacement (art. 17 RGPD) : Vous avez le droit de supprimer vos données, sauf si des obligations légales de conservation s'y opposent.
  • Limitation (art. 18 RGPD) : Vous avez le droit de demander la limitation du traitement de vos données.
  • Portabilité des données (art. 20 RGPD) : Vous avez le droit de recevoir vos données dans un format structuré, couramment utilisé et lisible par machine.
  • Opposition (art. 21 RGPD) : Vous avez le droit de vous opposer à tout moment au traitement pour des raisons liées à votre situation particulière.
  • Révocation du consentement : Vous pouvez révoquer un consentement accordé à tout moment.
